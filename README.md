# Maruti Suzuki Sales Dashboard (Power BI)

## Project Overview

This project is an interactive Power BI dashboard created to analyze Maruti Suzuki sales performance and business trends.

## Key KPIs

* Total Sales
* Total Revenue
* Monthly Growth %
* Top Selling Model

## Features

* Interactive slicers (Month, Region, Model)
* Sales trend analysis
* Region-wise revenue split
* Model-wise comparison
* Clean executive dashboard layout

## Tools Used

* Power BI
* DAX Measures
* Data Visualization
* Data Analysis

## Files Included

* Power BI (.pbix)
* Dashboard Screenshot
* Dataset
